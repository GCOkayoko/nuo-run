## 📖 关于

一款以《鸣潮》角色为主题的点击健身小游戏。

玩法很简单：在规定时间内尽可能多地点击屏幕，帮助糯糯完成健身目标。支持普通模式、无尽模式、练习模式，以及神秘的 🐱 模式。
